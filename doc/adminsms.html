adminsms Package¶

adminsms Module¶

Platform:

*nix
Synopsis:

adminssms - creates the daemon implemented in Daemon and listens to registered and authorized phone SMS notifications, executes commands in the SMS message and sends a reply with:

  • errors that might occur or
  • output of the command

If the output is more than 160 characters long, sends default message instead
